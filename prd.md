## 2. 게임 규칙

### 기본 흐름
1. 4개 색깔 버튼 (주황 / 파랑 / 초록 / 노랑)이 순서대로 깜빡임
2. 유저가 같은 순서로 버튼을 누름
3. 맞으면 다음 라운드 (기존 순서 + 1개 추가)
4. 틀리거나 2초 초과 시 게임 오버

### 난이도 선택

| 난이도 | 깜빡임 속도 | 점수 배율 |
|---|---|---|
| Easy | 0.5초 | x1 |
| Medium | 0.4초 | x2 |
| Hard | 0.3초 | x3 |

> 랭킹은 난이도 통합 운영 — Hard 유저가 유리한 구조가 정상  
> "랭킹 올리려면 Hard 도전" 이 자연스러운 동기 부여

### 타이머
- 버튼 하나당 입력 제한: **2초**
- 2초 내 미입력 = 즉시 게임 오버
- 타이머 UI: 버튼 주변 게이지바로 시각적 표시

### 점수 계산

**1~9스테이지**: 버튼 누를 때마다 **+1점** (클리어 보너스 없음)

**10스테이지 이상**: 버튼 +1 + 클리어 보너스 + 풀콤보 적용

| 스테이지 | 버튼 점수 | 클리어 보너스 | 합계 | 풀콤보(x2) |
|---|---|---|---|---|
| 10 | +10 | +2 | 12점 | 24점 |
| 15 | +15 | +3 | 18점 | 36점 |
| 20 | +20 | +4 | 24점 | 48점 |
| 25 | +25 | +5 | 30점 | 60점 |
| 30 | +30 | +6 | 36점 | 72점 |

> 클리어 보너스 = 스테이지번호 ÷ 5 (10스테이지 이상만)  
> 중간 탈락 시 맞춘 버튼 점수만 지급, 클리어 보너스 없음  
> **최종 점수 = 원점수 × 난이도 배율**

### 콤보 시스템 (10스테이지 이상)
- 해당 라운드의 **모든 버튼을 0.3초 이내** 연속 입력 → 풀콤보
- 풀콤보 달성 시 해당 스테이지 총점(버튼점수 + 클리어보너스) **x2**
- 한 개라도 0.3초 초과 시 기본 점수만 지급

### 난이도별 누적 점수 시뮬 (풀콤보 기준)

| 도달 스테이지 | Easy (x1) | Medium (x2) | Hard (x3) |
|---|---|---|---|
| 20 | 433 | 866 | 1,299 |
| 30 | 1,037 | 2,074 | 3,111 |
| 40 | 1,881 | 3,762 | 5,643 |
| 50 | 2,965 | 5,930 | 8,895 |

> Easy 50스테이지 ≈ Hard 30스테이지 (약 3,000점대)

## 3. 게임 기회 시스템

| 구분 | 횟수 | 조건 |
|---|---|---|
| 기본 기회 | 1회/일 | 자정 리셋 |
| 추가 기회 | 최대 3회/일 | 리워드광고 시청 1회당 +1 |

> 하루 최대 4회 플레이 가능

---

## 4. 랭킹 시스템

### 랭킹 종류 3개
| 구분 | 기준 | 리셋 |
|---|---|---|
| 일간 랭킹 | 당일 최고점수 | 매일 자정 |
| 월간 랭킹 | 월 최고점수 | 매월 1일 자정 |
| 시즌 랭킹 | 시즌 내 최고점수 | 3개월마다 (1/4/7/10월) |

### DB 구조
```sql
scores
- id
- user_id      -- 앱인토스 getUserKeyForGame().hash
- score
- played_at    -- 이걸로 일간/월간/시즌 필터링
```

## 6. 수익화 (BM)

### IAA (광고 수익)
| 광고 타입 | 노출 위치 | 비고 |
|---|---|---|
| 배너광고 | 게임 화면 하단 항상 노출 | |
| 전면광고 | 게임 오버 화면 | 스킵 가능 (5초 후) |
| 리워드광고 | 추가 기회 획득 버튼 클릭 시 | 유저 자발적 |

### IAP (직접 판매)
| 상품 | 가격 | 내용 |
|---|---|---|
| 광고 제거 | 1,990원 (1회성) | 배너 + 전면광고 제거. 리워드광고는 유지 |

## 7. 화면 목록 (MVP)

1. **메인 화면** — 게임 시작 버튼 + 남은 기회 표시 + 내 랭킹 순위
2. **게임 화면** — 4개 버튼 + 현재 점수 + 타이머 게이지 + 배너광고
3. **게임 오버 화면**
   - 이번 게임 점수
   - 랭킹 진입 시: 일간 / 월간 / 시즌 각 순위 표시
   - 랭킹 포인트 안내: "월간 XX위 → OO월 1일에 XXX원 지급 예정"
   - **한 번 더** 버튼 → 리워드광고 시청 → "광고를 보면 1회 추가됩니다. 계속하시겠습니까?" 확인 → 게임 재시작
   - 기회 소진 시: 한 번 더 버튼 비활성화 + "오늘 기회를 모두 사용했어요. 내일 다시 오세요"
4. **랭킹 화면** — 일간 / 월간 / 시즌 탭 + TOP 50 + 내 순위 하단 고정

## 8. MVP 개발 범위

### 포함
- 게임 코어 로직
- 타이머 (2초 제한)
- 점수 계산 + 콤보 시스템 (10스테이지~, 0.3초 이내 연속입력)
- 배너광고 + 리워드광고
- 랭킹 3종 (Supabase)
- 앱인토스 SDK 연동 (getUserKeyForGame)
- 난이도 선택 (Easy / Medium / Hard)

### 제외 (v2)
- 전면광고
- IAP (광고 제거)
- 랭킹 보너스 포인트 정산
- 시즌 랭킹 보너스
- 친구 초대 바이럴
