## 2. 게임 규칙

### 기본 흐름
1. 4개 색깔 버튼 (주황 / 파랑 / 초록 / 노랑)이 순서대로 깜빡임
2. 유저가 같은 순서로 버튼을 누름
3. 맞으면 다음 라운드 (기존 순서 + 1개 추가)
4. 틀리거나 2초 초과 시 게임 오버

### 난이도 선택

| 난이도 | 깜빡임 속도 | 점수 배율 |
|---|---|---|
| Easy | 0.5초 | x1 |
| Medium | 0.4초 | x2 |
| Hard | 0.3초 | x3 |

> 랭킹은 난이도 통합 운영 — Hard 유저가 유리한 구조가 정상
> "랭킹 올리려면 Hard 도전" 이 자연스러운 동기 부여

### 타이머
- 버튼 하나당 입력 제한: **2초**
- 2초 내 미입력 = 즉시 게임 오버
- 타이머 UI: 버튼 주변 게이지바로 시각적 표시

### 점수 계산

**1~9스테이지**: 버튼 누를 때마다 **+1점** (클리어 보너스 없음)

**10스테이지 이상**: 버튼 +1 + 클리어 보너스 + 풀콤보 적용

| 스테이지 | 버튼 점수 | 클리어 보너스 | 합계 | 풀콤보(x2) |
|---|---|---|---|---|
| 10 | +10 | +2 | 12점 | 24점 |
| 15 | +15 | +3 | 18점 | 36점 |
| 20 | +20 | +4 | 24점 | 48점 |
| 25 | +25 | +5 | 30점 | 60점 |
| 30 | +30 | +6 | 36점 | 72점 |

> 클리어 보너스 = 스테이지번호 ÷ 5 (10스테이지 이상만)
> 중간 탈락 시 맞춘 버튼 점수만 지급, 클리어 보너스 없음
> **최종 점수 = 원점수 × 난이도 배율**

### 콤보 시스템 (10스테이지 이상)
- 해당 라운드에서 **연속된 두 버튼 입력 간 간격이 모두 300ms 이하**일 때 풀콤보
- 풀콤보 달성 시 해당 스테이지 총점(버튼점수 + 클리어보너스) **x2**
- 한 gap이라도 300ms 초과 시 기본 점수만 지급

### 난이도별 누적 점수 시뮬 (풀콤보 기준)

| 도달 스테이지 | Easy (x1) | Medium (x2) | Hard (x3) |
|---|---|---|---|
| 20 | 433 | 866 | 1,299 |
| 30 | 1,037 | 2,074 | 3,111 |
| 40 | 1,881 | 3,762 | 5,643 |
| 50 | 2,965 | 5,930 | 8,895 |

## 3. 게임 오버 광고 흐름 ⚠️ v0.2 변경

> **기존 기회제 폐지** — 하루 플레이 횟수 제한 없음.
> 게임 오버 → 리워드광고 자동 재생 → 완시청 여부에 따라 포인트 지급 → 다시하기.

| 단계 | 행동 | 결과 |
|---|---|---|
| 게임 오버 | 리워드 광고 자동 시작 (강제) | 스킵 불가 상태로 광고 시작 |
| 완시청 (userEarnedReward) | 오늘 첫 완시청 여부 확인 | 첫 완시청이면 10포인트 지급, 이후는 포인트 없음 |
| 스킵 / 광고 실패 | 포인트 미지급 | 다시하기 버튼 활성화 |
| 다시하기 버튼 | 누르면 게임 재시작 | 플레이 횟수 제한 없음 |

> 오늘 첫 완시청 여부는 Supabase `daily_reward` 테이블로 판단 (자정 리셋)

## 4. 랭킹 시스템

### 랭킹 종류 3개
| 구분 | 기준 | 리셋 |
|---|---|---|
| 일간 랭킹 | 당일 최고점수 | 매일 자정 |
| 월간 랭킹 | 월 최고점수 | 매월 1일 자정 |
| 시즌 랭킹 | 시즌 내 최고점수 | 3개월마다 (1/4/7/10월) |

### DB 구조
```sql
scores
- id
- user_id      -- 앱인토스 getUserKeyForGame().hash
- score
- played_at    -- 이걸로 일간/월간/시즌 필터링
```

## 7. 화면 목록 (MVP) ⚠️ v0.2 변경

1. **메인 화면** — 게임 시작 버튼 + 내 랭킹 순위
2. **게임 화면** — 4개 버튼 + 현재 점수 + 타이머 게이지 + 배너광고
3. **게임 오버 화면**
   - 이번 게임 점수
   - 랭킹 진입 시: 일간 / 월간 / 시즌 각 순위 표시
   - 리워드 광고 자동 시작
   - 완시청 → 오늘 첫 완시청이면 "10포인트 지급!" 메시지 표시
   - 광고 종료 후 **다시하기** 버튼 활성화 (횟수 제한 없음)
4. **랭킹 화면** — 일간 / 월간 / 시즌 탭 + TOP 50 + 내 순위 하단 고정

## 8. MVP 개발 범위

### 포함
- 게임 코어 로직
- 타이머 (2초 제한)
- 점수 계산 + 콤보 시스템 (10스테이지~, 0.3초 이내 연속입력)
- 배너광고 + 게임오버 강제 리워드광고
- 랭킹 3종 (Supabase)
- 앱인토스 SDK 연동 (getUserKeyForGame)
- 난이도 선택 (Easy / Medium / Hard)
- 데일리 리워드 수령 기록 (Supabase `daily_reward`)

### 제외 (v2)
- 전면광고
- IAP (광고 제거)
- 랭킹 보너스 포인트 정산
- 시즌 랭킹 보너스
- 친구 초대 바이럴
